**🛡️ The Deadline Guardian**

An AI-powered productivity strategist built for the **Google AI Studio Vibe Coding Hackathon** under the **"The Last-Minute Life Saver"** problem statement.

The Deadline Guardian acts as an intelligent execution companion that helps users overcome procrastination, prioritize work, schedule focus sessions, build habits, and receive contextual AI coaching before deadlines are missed.

# AI Integration

The application uses **Google Gemini** as its intelligence engine.

The Guardian receives contextual information including:

- Active tasks
- Deadlines
- Habits
- Focus history
- Recent conversations
- Workspace status

This enables personalized productivity coaching instead of generic chatbot responses.
